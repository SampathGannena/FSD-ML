## Data Flow

```
User Action Flow:
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

1. User visits /Dashboards/group-recommendations.html
2. Frontend fetches JWT token from localStorage
3. Makes API request: GET /api/groups/recommendations?limit=10
4. Backend validates token â†’ extracts userId
5. Service queries MongoDB for:
   - User's current groups
   - All active groups
   - Filters out already-joined groups
6. For each candidate group, calculates:
   - Similarity score
   - Popularity score
   - Activity score
   - Availability score
   - Collaborative score
7. Combines scores with weights
8. Sorts by final score (descending)
9. Returns top N recommendations with reasons
10. Frontend displays in beautiful cards
11. User clicks "Join Group"
12. Frontend calls /api/save-current-group
13. User is added to group
14. Feedback recorded via /api/groups/recommendations/feedback
```
